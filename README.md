C++ Calculator

This is a simple calculator program written in C++.

 Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)

 How It Works

The program takes two numbers and an operator as input, then performs the calculation using a switch-case structure.

 How to Run

1. Compile the code:
   g++ calculator.cpp -o calculator

2. Run the program:
   ./calculator

Example

Enter first value: 5
Enter operator (+, -, *, /): +
Enter second value: 3

Output: 8
